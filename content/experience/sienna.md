# SIENNA — SysAdmin / Cloud-to-Tape Archiving

## Role
**Systems Administrator** — managing a specialized data archival platform bridging cloud object storage to physical tape libraries for long-term retention compliance.

## Responsibilities
- Administering Linux servers running the archival pipeline
- Maintaining tape library hardware (IBM TS3500 series)
- Automating data integrity verification workflows in Python/Bash
- Incident response for data pipeline failures

## Technical Stack
`LTFS`, `Tape libraries`, `Object storage (S3-compatible)`, `Python`, `Bash`, `Linux`, `Monitoring (Prometheus)`
